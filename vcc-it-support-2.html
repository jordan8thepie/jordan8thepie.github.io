Project Overview

I was tasked with a complete technical overhaul of the Victory Christian Centre Trust’s multi-building campus. I transitioned the organization from a high-risk "Shadow IT" environment to a professional, secure, and automated Enterprise ecosystem.

Key Achievements
  • 100% Uptime for Live Broadcasts
  • 90% Reduction in Device Setup Time
  • Zero-Loss Data Migration
  • Full Campus Network Visibility

Phase 1: Identity & Data Sovereignty

Entra ID Identity Migration

Replaced fragmented local user accounts with centralized Microsoft Entra ID identities. This enabled Single Sign-On (SSO) and enforced Multi-Factor Authentication (MFA) across the entire staff fleet.

SharePoint Data Governance

Remediated a high-risk data structure where the "Server" was hosted on a personal OneDrive. Migrated all files to SharePoint Online with Role-Based Access Control (RBAC) using M365 Security Groups.

Phase 2: Network & Physical Infrastructure

Core Gateway

Deployed a UniFi Dream Machine SE and PoE Switches, providing full Layer 3 switching and network-wide threat management.

L1 Infrastructure

Restored Structured Cabling in the secondary building, terminating runs into a new 24-port patch panel to "liven" the site.

Asset Lifecycle

Managed the procurement and refurbishment of 2nd-hand hardware, configured via Intune/Autopilot for zero-touch deployment.

Phase 3: Media & Hybrid Operations

Live Broadcast & Streaming

Managed the end-to-end technical production of weekly live broadcasts. Coordinated audio-visual sync, camera switching, and digital graphics via ProPresenter.

Sound Reinforcement
Multi-Camera Op
Hybrid Teams Room

Engineered a budget-friendly Microsoft Teams Room using a mix of refurbished hardware and professional peripherals to enable seamless remote collaboration.

Lighting System Integration

Acted as the Technical Project Lead for the installation of a new stage lighting rig. Managed vendor relationships, coordinated with the lighting designer, and oversaw the physical system commissioning and DMX patching.
